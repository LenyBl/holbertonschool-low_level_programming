# more_malloc_free

Suite du module malloc/free : exercices avancés. - Utilisation de
`malloc`, `calloc`, `realloc` - Gestion de tableaux multidimensionnels -
Manipulation de mémoire dans des structures plus complexes

## Compilation

``` bash
gcc -Wall -Wextra -Werror -pedantic -std=gnu89 <fichier>.c -o <executable>
```

## Objectifs

-   Comprendre les limites de `malloc`
-   Utiliser `realloc` correctement
-   Éviter les memory leaks dans des fonctions plus complexes
