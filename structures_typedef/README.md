# structures_typedef

Ce dossier présente les bases de l'utilisation des structures en C et du
mot‑clé `typedef`.

## Concepts abordés

-   Déclaration de structures
-   Accès aux champs d'une structure
-   Pointeurs sur structures
-   Définition de types personnalisés avec `typedef`

## Compilation

``` bash
gcc -Wall -Wextra -Werror -pedantic -std=gnu89 <fichier>.c -o <executable>
```

## Objectifs

-   Structurer les données complexes
-   Comprendre l'intérêt de `typedef`
-   Manipuler des structures dynamiques avec `malloc`
